FIFA 19 Dataset of all Players in the Game

Data was gathered from the latest EA FIFA game, FIFA 19, to give all stats of current players in the game. The data can be used to find players that meet specific queries, which may not be possible to find witout a dataset like this.
Link to the data file and more about it can be found here.
